App to find cash level (L1,L2,L3). <br />
To start install req: matplotlib for python.<br />
How to start:
```
chmod +x run.sh
./run.sh
```
To check your L1-L3:
```
 lscpu | egrep cache
```

My result you can find on pic folder<br />
I ran on Intel i5-4210u<br />
L1d cache:           32K<br />
L1i cache:           32K<br />
L2 cache:            256K<br />
L3 cache:            3072K<br />
