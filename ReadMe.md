App to find cash level (L1,L2,L3). 
To start install req: matplotlib for python.
How to start:
```
chmod +x run.sh
./run.sh
```
To check your L1-L3:
```
 lscpu | egrep cache
```

My result you can find on pic folder.
I ran on Intel i5-4210u
L1d cache:           32K
L1i cache:           32K
L2 cache:            256K
L3 cache:            3072K
